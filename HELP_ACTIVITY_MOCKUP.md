## User Experience Flow

1. **User opens MainActivity**
   - Sees Emergency and Panic mode buttons
   - Each has an info icon (â“˜) next to it

2. **User taps info icon next to "Activate Emergency Communication"**
   - HelpActivity opens
   - Tab "Emergency" is pre-selected
   - User sees detailed flow diagram

3. **User can swipe left/right or tap tabs**
   - Navigate between 4 different scenarios
   - Each shows a visual flow diagram
   - Monospace font ensures diagrams align properly

4. **User taps back button**
   - Returns to MainActivity
   - Can activate emergency mode with better understanding
